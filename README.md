# 🇪🇸 Español

Aplicación de escritorio en Python que permite comparar dos archivos de texto (`.txt`) mediante una interfaz gráfica creada con **Tkinter**.  
El programa genera automáticamente un archivo **`diferencias.txt`** con:

- 📌 Líneas presentes solo en el archivo 1  
- 📌 Líneas presentes solo en el archivo 2  

Ideal para comparar listas, identificadores, registros o cualquier conjunto de líneas.

# 🇬🇧 English

Desktop Python application that enables comparison between two text files (`.txt`) through a graphical interface built with **Tkinter**.  
The program automatically generates a **`diferencias.txt`** file containing:

- 📌 Lines present only in file 1  
- 📌 Lines present only in file 2  

Perfect for comparing lists, identifiers, records, or any set of text lines.
